# Example stepped wedge data

Data generated for the purpose of demonstrating the steppedwedge package

## Usage

``` r
sw_data_example
```

## Format

\## \`sw_data_example\` A data frame with 2,063 rows and 5 columns:

- cluster:

  Cluster id

- period:

  Time period

- trt:

  Treatment indicator

- outcome_bin:

  Binary outcome

- outcome_cont:

  Continuous outcome
